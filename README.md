# Expense Tracker

## Project Overview

Expense Tracker is a user-friendly web app built with React.js that helps people keep track of their income and expenses. It provides a simple way to manage personal finances by allowing users to add, edit, and delete transactions, categorize their spending, and see real-time updates on their balance. This project was created as part of an internship with Aarna Tech by a team of three interns.

---

## Key Features

- Easily add, edit, or remove income and expenses  
- Organize transactions by categories  
- See your current balance along with total income and expenses updated in real-time  
- Responsive design that works well on both desktop and mobile  
- Clean, intuitive interface focused on user experience
